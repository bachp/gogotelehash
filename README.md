## Project Status

Gogotelehash is currently under active development. The API is subject to change. We are currently working on Telehash v3.

What works:

* Endpoints
* Exchanges
* Channels (both reliable and unreliable)
* cipherset 1a
* cipherset 3a
* transport udp
* transport inproc
* upnp and nat-pmp mapping
